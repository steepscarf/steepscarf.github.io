CONTROLS:
---------

Z: Run / Fire / Pickup / Select level
X: Jump
Enter: Pause / Select
Arrow Keys: Move / Move Menu Options

HOW TO ACCESS PLUS MODE:
------------------------

Collect all the Special Coins in all of the levels, with the exception of Mario's House AKA the prologue level (the Special Coins don't count there) and Air Man's stage (that stage has no Special Coins at all).

NOTES:
------

This is the feedback demo, so there may be some oversights/bugs in certain areas.

If any bugs occur, please send me the info so I can fix those up.
